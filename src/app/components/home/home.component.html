Регистрация установки 1. Перейдите в раздел "Установки"; 2. Зарегистрируйте установку, если его нет в списке. Нужно добавить через кнопку "Создать"; 3. Выделите нужную установку; 4. Внесите данные в разделе "Описание процессов". Через кнопку "Создать;"    4.1. Выберите основной производственный процесс. Если его нет в списке можно добавить через кнопку "Добавить, если нет в списке";    4.2. Перечислите список подпроцессов. Если его нет в списке можно добавить через кнопку "Добавить, если нет в списке";    4.3. Перечислите список сырья. Если его нет в списке можно добавить через кнопку "Добавить, если нет в списке"; 5. Внесите данные в разделе "Перечень продукции". Через кнопку "Создать";    5.1. Укажите вид продукции. Если его нет в списке можно добавить через кнопку "Добавить, если нет в списке";    5.2. Укажите единицу измерение. Если его нет в списке можно добавить через кнопку "Добавить, если нет в списке"; 6. Внесите данные в разделе "Сфера деятельности". Через кнопку "Создать";    5.1. Укажите основную сферу и под сферу деятельности;    5.2. Укажите единицу измерение. Если его нет в списке можно добавить через кнопку "Добавить, если нет в списке";
